Walkthrough

What the reviewer can verify.

  • Open the portfolio home page and navigate to the Systematic Trading Framework project.
  • Review the project overview, backtest summary, robustness checks, diagnostics, and implementation notes.
  • Inspect the GitHub repository link for source code and project context.
  • Confirm that the public site provides privacy, terms, support email, and project contact links.
  • Confirm that the website does not request user accounts, payments, brokerage credentials, or personal financial data.

Demo Notes

The public website is a static portfolio and research demonstration. It does not include authenticated user flows, in-app purchases, subscriptions, digital goods, or live trading execution.

If a review form requires an actual video file, use this page as the walkthrough script and upload a short recording to a stable public URL such as YouTube, Loom, Google Drive with public access, or a hosted MP4.

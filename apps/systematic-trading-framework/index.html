Inspect trading research repositories safely

Capabilities

What the app does

The app exposes safe MCP tools for reading and understanding a repository explicitly mounted by the user. It supports file reads, directory listings, code search, symbol search, reference lookup, project tree inspection, git status and diff review, configuration and log reading, experiment artifact review, and Optuna SQLite database inspection.

The server is intended for local and private development workflows. It is designed to avoid access outside the mounted repository and should not be configured to expose a user's home directory or whole computer.

Supported workflows

  • Inspect source files, Markdown documentation, notebooks, YAML, JSON, TOML, CSV, SQL, Docker files, and logs.
  • Search repository content and find code symbols or references.
  • Review git status, current branch, diffs, and recent commit history.
  • Read experiment results, configuration files, logs, and Optuna SQLite databases.
  • Run only approved repository Python scripts when the user explicitly confirms execution.

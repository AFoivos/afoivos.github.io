Privacy Policy

This policy describes how the Systematic Trading Framework ChatGPT MCP app handles repository data and user information.

Last updated: June 27, 2026

Overview

Systematic Trading Framework is a private/local MCP-based developer tool. It lets ChatGPT inspect and analyze a repository explicitly mounted by the user, typically a local systematic trading research repository.

Repository Access

The app accesses only repositories that the user explicitly mounts or connects to the MCP server. The app should not be configured to expose a user's whole computer, home directory, unrelated folders, or secrets outside the intended repository.

Data Processed

Depending on the user's request, the app may process repository files, source code, configuration files, Markdown documentation, notebooks, CSV files, logs, git metadata, experiment artifacts, and Optuna SQLite databases available inside the mounted repository.

Use of Data

Repository data is used to answer the user's requests in ChatGPT, such as reading files, searching code, explaining configuration, summarizing logs, inspecting git state, or reviewing experiment results. The app does not collect personal data for resale and does not sell repository data.

Execution and Writes

Read-only tools can inspect repository content without additional confirmation. Any write or execution capability must be limited, allowlisted, and require explicit user confirmation before running. The app does not provide arbitrary shell execution.

Financial and Trading Data

The app is a developer tool for repository inspection. It does not execute live trades, place broker orders, manage brokerage accounts, process payments, or provide financial advice.

Third-Party Processing

When used with ChatGPT, prompts, tool calls, and tool outputs are processed by OpenAI according to the applicable OpenAI terms and privacy controls for the user's account or workspace. If the user hosts the MCP server or tunnel through another provider, that provider may process network metadata according to its own policies.

Data Retention

The MCP server is intended for local/private development use and does not maintain a separate user account database for resale or advertising. Repository data may appear in the user's ChatGPT conversation history depending on the user's OpenAI settings and workspace policies.

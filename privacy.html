Overview

This is a static portfolio website hosted on GitHub Pages. It presents professional background, project summaries, research artifacts, and contact links. This website does not include custom analytics, tracking pixels, contact forms, cookies, account registration, or databases controlled by Foivos Georgios Ampatzis.
